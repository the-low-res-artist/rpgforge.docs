## Feature details

### New Unity Scene
A new Unity Scene is create with a few GameObjects :
* **Root** : A root gameobject for the whole Scene.
  * **Tilemap** : A Tilemap gameobject to start drawing tiles.
  * **Start** : A Spawn gameobject for the Player character.
